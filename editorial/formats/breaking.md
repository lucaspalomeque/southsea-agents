# Formato: Breaking
## Noticia urgente — rápida, factual, con contexto mínimo

---

## Cuándo usar
- Algo acaba de pasar y es relevante para el ecosistema
- No hay tiempo para análisis profundo
- El valor es la velocidad + contexto mínimo necesario

## Largo
200-400 palabras

## Estructura

```
# [Qué pasó — en una oración]

[El hecho]
Qué, quién, cuándo, dónde. 2-3 oraciones factuales.
Solo datos verificados del brief.

## Contexto rápido
¿Por qué importa? 1-2 párrafos.
Lo mínimo necesario para que el lector entienda
la relevancia sin tener que buscar más.

## Qué mirar
¿Qué sigue? ¿Qué hay que observar?
1-2 oraciones de cierre.
```

## Notas
- Velocidad sobre profundidad
- CERO especulación — solo hechos verificados
- Si no hay suficientes datos verificados, no publicar como breaking
- Se puede expandir a analysis después si el tema lo amerita
